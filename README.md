# YouOffline

Android-приложение для скачивания аудио и видео с YouTube и других сайтов для offline-прослушивания/просмотра.

## Возможности

- **Скачивание аудио** — сохраняет в формате M4A в папку «Музыка»
- **Скачивание видео** — до 1080p, сохраняет в папку «Видео»
- **Плейлисты** — скачивает треки по одному с отображением прогресса по каждому
- **Библиотека** — просмотр и воспроизведение скачанных файлов прямо в приложении
- **Поделиться** — можно передать ссылку из браузера через «Поделиться»

## Установка

1. Скачай APK из раздела [Releases](https://github.com/AlexShkryabin/YouOffline/releases)
2. На телефоне разреши установку из неизвестных источников:  
   **Настройки → Безопасность → Установка неизвестных приложений**
3. Открой скачанный APK и установи

> Требуется Android 7.0 (API 24) или новее.

## Использование

### Скачать аудио
1. Скопируй ссылку на видео или плейлист YouTube
2. Открой YouOffline → вкладка **Аудио**
3. Вставь ссылку и нажми **Скачать**

### Скачать видео
1. Перейди на вкладку **Видео**
2. Вставь ссылку, выбери качество и нажми **Скачать**

### Через «Поделиться»
В браузере нажми «Поделиться» → выбери **YouOffline** → начнётся загрузка аудио.

## Сборка из исходников

1. Скопируй `ads.properties.example` → `ads.properties` и укажи свои рекламные Unit ID (или оставь заглушки — реклама просто не будет показана в debug-сборке)
2. Открой проект в Android Studio
3. **Build → Make Project**

## Технологии

- [yt-dlp](https://github.com/yt-dlp/yt-dlp) — движок загрузки
- [youtubedl-android](https://github.com/yausername/youtubedl-android) — обёртка yt-dlp для Android
- [ExoPlayer (Media3)](https://developer.android.com/media/media3) — воспроизведение
- Jetpack Compose + Material 3 — интерфейс
